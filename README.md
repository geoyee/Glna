# 图形化水准网平差软件

基于C#的图形化数据输入的水准网平差系统，可用于处理经典水准网、秩亏自由网以及带粗差的水准网。并添加了模拟网生成、图形化输入等扩展功能。

## 使用方式

1. 说明视频参照MP4下的四个说明文件。

2. 可执行的文件位于g-lna\s.ad_gp\s.ad_gp\bin\Debug下的s.ad_gp.exe。
